How It Works
MFA Setup Decision Process:
  1. Check Session Risk Score
    • If risk ≥ 60: BLOCK
    • Based on: impossible travel, new device, new country
  2. Check Recent Impossible Travel
    • Look back 24 hours
    • If found: BLOCK
  3. Check Active Alerts
    • Look for critical/high severity alerts
    • If found: BLOCK
  4. All Checks Pass
    • ALLOW MFA setup
    • Log the decision
This prevents attackers from adding their own MFA methods after compromising an account.
